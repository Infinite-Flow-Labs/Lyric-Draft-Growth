# TEMPLATE_RULES.md — xiaodao-ai-lab

## Global Writing Constraints
- 标题：冲突 + 结果
- 开头：前两句必须有反差/代价/变化
- 结构：轻结构可读，不写流水账
- 字数：较基础版更丰富（建议 320–520 字）

## Hidden 3 Elements (must appear)
1) 案例细节（真实执行片段）
2) 量化对比（前后变化）
3) 复用方法（可直接照做）

## Axis by Template
- A: 过程主轴（案例最重）
- B: 数据主轴（结果最重）
- C: 方法主轴（角色与杠杆）
- D: 方法主轴（体验边界）
- E: 方法主轴（流程复用）

## Structure Rotation
- V1: 冲突开头 → 案例 → 数据 → 结论
- V2: 结果开头 → 反推过程 → 坑点 → 方法
- V3: 失败开头 → 修正动作 → 结果 → 复用

## Style Notes
- 少用“第一/第二/第三”
- 多用复盘语气："我这次踩坑是…/后来我改成…/真正有用的是…"
