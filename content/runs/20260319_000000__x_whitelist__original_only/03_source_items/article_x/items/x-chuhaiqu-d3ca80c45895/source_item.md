Post title: Stripe 刚推的 MPP（机器支付协议），简单来说，就是让 AI 也能自己掏钱了。 对现成的 SaaS 来说，吃下这波新营收并不难。 底层的提现和风控还是用你原来那套 Stripe 设施，核心只要把遇到付费墙时的 401 Unauthorized 报错，换成一个机器能读懂的支付请求。 跑通之后，你的客户群就多了一大批不需要客服的 Agent： 比如你做了一个深度行业数据库。Claude 帮人做调研时遇到付费节点，自己花几美分调用 API 买下数据，顺滑地把报告写完。 再比如你提供基础设施服务。OpenClaw 跑自动化任务走到一半，自动按次扣费，租用你的无头浏览器接着跑。 一手交钱一手交货，好好好，Agent 经济的闭环拼图又补上了一大块 🤣
