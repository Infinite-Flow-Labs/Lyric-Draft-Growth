Editor 16 March 2026 With Rotate Object, you can rotate, spin, tilt, or dolly an object from a pixel layer. Adobe has introduced a new function to Photoshop that allows users to rotate 2D objects, transforming them into 3D images. The feature is available in the Photoshop beta version 27.5.0. The new Rotate Object feature lets users rotate, spin, tilt, or dolly a 2D object from any viewpoint to match perspective and depth. The function fits into a standard Photoshop workflow. To get started, open or place an image and select a pixel layer containing the desired object. To make it easier to preserve the original and make adjustments later, it is recommended to duplicate the layer first, as it will be converted to a Rotate Object Layer. After, press Cmd + T to enter transform mode or Edit > Rotate Object, then click the Rotate object button in the context bar.
