Post title: You can now teach a humanoid robot tennis without showing it a single real match. LATENT is a new paper from Tsinghua University. It skips recording perfect game footage entirely. Instead, it uses short, messy clips of basic moves. A forehand swing here, a sidestep there. Just pieces, not full matches. A two-layer system does the heavy lifting: 1. The bottom layer watches clips and learns individual moves. 2. The top layer figures out when to use which move and fixes mistakes on the fly. It runs on a Unitree G1 robot. The machine holds rallies with real humans. Balls fly in at 15-30 m/s. It reacts within milliseconds. What made this possible: > No clean training data required > No real match recordings needed > No human remote-controlling the robot > Works outside simulation too The same approach could work for soccer and parkour. Any sport where collecting clean data is nearly impossible.
