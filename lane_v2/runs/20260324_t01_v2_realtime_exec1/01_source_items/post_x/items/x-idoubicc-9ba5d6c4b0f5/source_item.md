Post title: 微信上线了一个 ClawBot 插件，连接自己的 OpenClaw，作为联系人对话，日常使用很方便。 如何开始： 1. 手机微信更新到最新版本 2. 微信 App 关闭重启（不是退出登录） 3. 进入微信-我-设置-插件页面，查看 ClawBot 插件详情，点击连接 4. 在安装了 OpenClaw 的电脑上执行命令，安装微信插件👇 npx -y @tencent-weixin/openclaw-weixin-cli@latest install 5. 打开微信扫描上一步输出的二维码，登录微信 6. 微信对话列表出现一个 微信ClawBot 联系人，开始对话 几点注意： 1. 微信ClawBot 插件正在灰度，可能你更新完微信，也不一定能看到这个插件，需要等等 2. 跑在个人电脑的 OpenClaw 可以扫码连接，跑在云服务器或者托管服务的 OpenClaw 可能不支持扫码连接 3. 目前一个微信，最多开启一个 微信ClawBot，最多连接一个 OpenClaw 4. 微信ClawBot 跟文件传输助手一样是一级入口，目前不支持群聊，不支持转发消息给 ClawBot 5. 默认情况下，连接完 OpenClaw，你的微信 ClawBot 只能文字对话，如果要支持图片识别，需要在你的 OpenClaw 接入支持多模态的模型（比如 claude） 6. 如果要让微信ClawBot 生成图片、音乐等，需要你在 OpenClaw 自行配置 Skills，比如你可以把 nano banana 的 API 文档通过对话发给 ClawBot，让它自己写个 Skills 来调用，你只需要给他一个 API Key 7. 目前微信 ClawBot 不支持设置定时任务主动推送消息给你，需要你主动发消息对话 ------ 有了微信ClawBot 这个一级入口，日常使用起来非常方便，连接的又是自己的 OpenClaw，能记住你的偏好，能自定义各种能力，这一次是真的能让每个人拥有一个 AI 助理了。 搞 AI 还是得看龙哥，微信传输助手可以卸载了🤣
